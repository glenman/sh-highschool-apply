# 上海高中学校数据结构说明

## 数据来源

本数据来源于 `d:\Workspace\high-school-apply\dataref\2025上海高中学校排名名单.xlsx` 文件，包含了2025年上海市所有高中学校的详细信息。

## 数据结构

数据以JSON格式存储在 `d:\Workspace\high-school-apply\public\data\shanghai_high_schools.json` 文件中，包含一个学校对象数组，每个学校对象包含以下字段：

### 基本信息

| 字段 | 类型 | 描述 | 示例值 |
|------|------|------|--------|
| `学校名称` | 字符串 | 学校的名称 | "复旦附中" |
| `所属区` | 字符串 | 学校所在的行政区 | "全市" 或 "徐汇区" |
| `学校层次` | 字符串 | 学校的层级标签 | "四校"、"八校"、"市重点"、"区重点"等 |

### 2025年招生数据

| 字段 | 类型 | 描述 | 示例值 |
|------|------|------|--------|
| `25年到区` | 数字 | 2025年名额分配到区的招生人数 | 255.0 |
| `25年到区加权均分` | 数字 | 2025年名额分配到区的加权平均分 | 710.18 |
| `25年到校` | 数字 | 2025年名额分配到校的招生人数 | 62.0 |
| `25年到校加权均分` | 数字 | 2025年名额分配到校的加权平均分 | 699.01 |
| `25年统一` | 数字 | 2025年统一招生的招生人数 | 17.0 |
| `25年统一加权均分` | 数字 | 2025年统一招生的加权平均分 | 706.82 |
| `25年三批次加权均分` | 数字 | 2025年三个批次的加权平均分 | 707.94 |

### 比较数据

| 字段 | 类型 | 描述 | 示例值 |
|------|------|------|--------|
| `25VS24加权均分` | 数字 | 2025年与2024年的加权平均分对比 | 3.33 |
| `25生源质量位次` | 数字 | 2025年生源质量的位次 | 1.0 |
| `24生源质量位次` | 数字或字符串 | 2024年生源质量的位次 | 2 或 "S" |

## 数据类型说明

- **数字类型**：包含整数和浮点数，用于表示招生人数和分数
- **字符串类型**：用于表示学校名称、行政区和学校层次
- **特殊值**：
  - `NaN`：表示缺失的数值数据
  - `"S"`：表示特殊位次（可能是指顶级学校）

## 学校层次分类

| 层次标签 | 描述 |
|---------|------|
| 四校 | 上海市顶尖的四所高中：复旦附中、上海中学、交大附中、华师二附 |
| 八校 | 上海市传统的八所重点高中 |
| 四分 | 四校的分校或附属学校 |
| 新五虎 | 近年来表现突出的五所高中 |
| 委属市重 | 上海市教委直属的市重点高中 |
| 普通市重 | 普通的市级重点高中 |
| 公办区重 | 公办的区级重点高中 |
| 民办区重 | 民办的区级重点高中 |
| 公办特高 | 公办的特色高中 |
| 公办普高 | 公办的普通高中 |
| 民办普高 | 民办的普通高中 |

## 招生范围说明

| 所属区 | 招生范围 |
|-------|----------|
| 全市 | 面向全市招生 |
| 各区名称 | 主要面向本区招生，可能也会面向其他区少量招生 |

## 数据使用建议

1. **查询功能**：可以根据学校名称、行政区、学校层次等字段进行查询
2. **排序功能**：可以根据加权平均分、生源质量位次等字段进行排序
3. **分析功能**：可以分析不同区域、不同层次学校的招生情况和生源质量
4. **可视化功能**：可以将数据可视化，展示学校排名、分数分布等
