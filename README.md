# 上海中考志愿填报系统

基于中考总分的智能志愿推荐系统，帮助上海中考考生科学填报志愿。

## 功能特点

- **智能推荐**：基于用户输入的预估中考总分，生成个性化的志愿推荐
- **三批次推荐**：提供名额分配到区、名额分配到校和统一招生三个批次的推荐
- **梯度合理**：按照冲稳保梯度生成推荐，确保志愿填报科学合理
- **数据真实**：基于2025年上海市高中学校录取数据，确保推荐结果准确可靠
- **界面友好**：简洁明了的界面设计，操作简单直观

## 使用方法

1. **填写基本信息**：
   - 选择所在行政区
   - 选择初中性质（公办/民办）
   - 输入预估中考总分（0-750分）
   - 选择志愿偏好（冲刺/稳妥/保底）

2. **生成志愿推荐**：
   - 点击"生成志愿推荐"按钮
   - 系统会根据您的信息生成推荐结果

3. **查看推荐结果**：
   - 名额分配到区：1个志愿
   - 名额分配到校：2个志愿
   - 统一招生平行志愿：15个志愿

## 推荐规则

### 统一招生平行志愿（15个）

- **冲刺学校**：分数高于预估中考总分+20
- **稳妥学校**：分数在预估中考总分-5到预估中考总分+5之间
- **保底学校**：分数低于预估中考总分-35

### 名额分配到区（1个）

- 不低于预估中考总分
- 根据志愿偏好调整推荐范围

### 名额分配到校（2个）

- 分数在预估中考总分-10到预估中考总分+20之间
- 避免推荐分数过高的学校

## 技术栈

- **前端框架**：React + TypeScript
- **构建工具**：Vite
- **样式框架**：Tailwind CSS
- **数据格式**：JSON

## 数据来源

学校数据来源于上海市教育考试院2025年高中学校录取数据，包含278所高中的详细信息。
