## 一、核心录取批次优先级（程序执行顺序基准）

**录取顺序不可逆，前序批次录取后，后续志愿自动失效**，这是程序必须遵守的核心流程规则：

```mermaid

A[自主招生录取] --> B[名额分配综合评价录取] --> C[统一招生录取]
```
### 子批次内部优先级（同批次内的录取顺序）

1. 自主招生子批次顺序：

    1. 市实验 / 市特色高中普通自招

    2. 市实验 / 市特色高中体艺自招

    3. 国际课程班 / 中外合作办学自招

    4. 中职校自招（中本贯通→五年一贯制 / 中高职贯通→中职校提前招生）

2. 名额分配子批次顺序：

    1. 名额分配到区

    2. 名额分配到校

3. 统一招生子批次顺序：

    1. 1-15 平行志愿

    2. 征求志愿

## 二、各批次志愿配置规则（程序表单 / 志愿池基准）

### 2.1 自主招生志愿（可兼报所有类别）

|志愿类别|志愿数量|志愿模式|填报资格限制|程序逻辑|
|---|---|---|---|---|

|中职校提前招生|3 个|平行志愿|需通过对应校面试 / 测试 / 体艺资格|仅开放学生已通过测试的专业志愿|
### 2.2 名额分配综合评价志愿（可兼报）

|志愿类别|志愿数量|志愿模式|填报资格限制|程序逻辑|
|---|---|---|---|---|
|名额分配到区|1 个|顺序志愿|所有报名学生均可填报|志愿池仅保留：**在学生报名所在区投放了到区计划的高中**|

## 三、资格校验规则（程序前置过滤，剔除无效志愿）

### 3.1 核心资格判断逻辑（程序入口校验）

```python

# 伪代码：名额分配到校资格判断
def check_school_quota_eligibility(student):
    if student.is_graduate: return False # 往届生无资格
    if student.is_return_sh: return False # 返沪生无资格
    if student.is_cross_district: return False # 跨区报名无资格
    if student.middle_school.is_choose_student: return False # 选择生源初中（民办）无资格
    if student.study_years < 3: return False # 在籍不满3年无资格
    return True
```

### 3.2 志愿池过滤规则

1. **名额分配到区志愿池**：

    - 输入：学生报名区

    - 过滤：仅保留高中→该生所在区有投放「到区招生计划」的学校

    - 排除：未在本区投放计划的市实验性高中

2. **名额分配到校志愿池**：

    - 输入：学生所在初中

    - 过滤：仅保留该初中**分配到的到校名额高中**（每所公办初中的到校名额不同）

    - 排除：其他初中的到校名额学校

3. **统一招生志愿池**：

    - 输入：学生报名区

    - 过滤：仅保留在本区有统一招生计划的学校 / 专业

## 四、录取 / 排序规则（程序位次匹配 / 同分排序基准）

### 4.1 名额分配综合评价录取总分规则

名额分配批次的录取总分 = 学业考试总成绩（750 分） + 综合素质评价成绩（50 分）

> 综合素质评价默认赋 50 分，仅不合格时扣分，程序中默认按 + 50 分计算，除非学生评价不合格
> 
> 

### 4.2 同分排序规则（所有批次末位同分通用）

当投档末位出现同分，按以下优先级排序，程序中排序字段必须严格按此顺序：

```python

# 同分排序优先级（从高到低）
sort_priority = [
    "has_same_score_preference", # 1. 同分优待（现役军人/消防子女）
    "quality_evaluation_score",  # 2. 综合素质评价成绩
    "chinese_math_english_sum",  # 3. 语数外三科总分
    "math_score",                # 4. 数学单科成绩
    "chinese_score",             # 5. 语文单科成绩
    "comprehensive_test_score"   # 6. 综合测试成绩
]
```

### 4.3 平行志愿录取规则

所有平行志愿（中职自招、到校志愿、1-15 志愿）均遵循：

> **分数优先、遵循志愿、一轮投档**
> 程序中按学生总分从高到低排序，依次匹配学生志愿，一旦匹配到有剩余计划的学校即投档，不再处理后续志愿
>

## 五、特殊政策规则（程序特殊处理逻辑）

### 5.1 政策性照顾加分规则

1. 加分规则：1-8 项加分**不可累计**，仅取最高项

2. 加分映射（程序加分逻辑）：
| 照顾类型 | 加分值 | 适用批次 |
|---------|-------|---------|
| 烈士子女、三类艰苦地区军人子女等 | 20 分 | 所有批次 |
| 作战部队军人子女、因公牺牲民警子女等 | 10 分 | 所有批次 |
| 归侨 / 华侨子女、少数民族学生、台湾学生 | 5 分 | 所有批次 |
| 其他现役军人 / 消防子女 | 同分优待（不加分，仅同分排序优先） | 名额分配 / 统一招生批次 |

### 5.2 最低投档控制分数线规则

所有批次录取必须满足最低线，程序中推荐志愿时需过滤掉学生成绩达不到最低线的学校：

|批次|最低控制线|
|---|---|
|公办高中自招、名额分配综合评价|全市统一最低线|
|民办国际课程班自招|学校自主设定最低线（≥市自招线）|
|中本贯通|普通高中最低投档线|
|五年一贯制 / 中高职贯通|对应专业最低线|
### 5.3 外籍学生特殊规则

持外国护照报名的学生：

- 不可填报所有批次志愿，不参与正常录取

- 由区教育局统一安排入学，程序中需单独区分该类学生
