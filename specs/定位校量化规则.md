# 上海中考志愿填报·定位校量化规则（完整版）
## 一、核心定义（量化基础）
### 1. 定位校 L
- 唯一基准：**统一招生15个平行志愿**内确定
- 量化依据：**一模/二模区内位次**匹配近3年各校录取位次
- 取值：**L = 层次相近2–3所学校的中位校**（记为1个基准）
- 分数量化：设定位校对应**基准分 = L分**

### 2. 学校层级量化（统一标尺）
- 冲刺校：**≥ L分 + 5分**
- 稳妥校：**L分 ± 5分**
- 保底校：**≤ L分 – 5分**
- 梯度差：相邻志愿**≥3分**，首尾保底**≥15分**

### 3. 考生类型量化
- A类（稳定）：模考位次差**≤5%**
- B类（波动）：模考位次差**5%–15%**
- C类（偏弱）：模考位次差**>15%**

---

## 二、平行志愿（15个）量化选择规则
### 1. 7+1+7 量化结构
1. **定位位（第8位）**：强制填**定位校L**
2. **前7位（冲刺）**
   - A类：**L+5 ~ L+20**
   - B类：**L+5 ~ L+15**
   - C类：**L+5 ~ L+10**
3. **后7位（保底）**
   - A类：**L–5 ~ L–25**
   - B类：**L–5 ~ L–30**
   - C类：**L–5 ~ L–35**

### 2. 强制量化约束
- 必须填满**15个**
- 第1–15位**严格从高到低**，分数单调递减
- 定位校前后各3所**梯度≤5分**，两端**梯度≥5分**
- 最后2所保底：**近3年录取位次≥考生位次+20%**

### 3. 录取量化规则
- 分数优先、遵循志愿、一轮投档
- 一旦录取，后续志愿全部失效

---

## 三、名额分配到校 量化选择规则
### 1. 资格量化
- 满足：**不择生源初中 + 在籍满3年**
- 不满足：**禁止填报**

### 2. 分配校分值 D
- 用近3年录取分量化为**D分**

### 3. 填报判定（量化阈值）
1. **D ≥ L + 5** → **必须填**（冲优）
2. **L – 5 < D < L + 5** → **建议填**（稳妥）
3. **D ≤ L – 5** → **不填**（提前锁低分）

### 4. 志愿排序量化
- 2个平行志愿：**D1 ≥ D2**（从高到低）
- 录取：校内排序，先于平行志愿

---

## 四、名额分配到区 量化选择规则
### 1. 基础规则
- 仅**1个志愿**，全区排序录取
- 底线：**不低于定位校L**

### 2. 按考生类型量化填报
- **A类（稳定）**：**Q = L + 10 ~ L + 20**
- **B类（波动）**：**Q = L + 0 ~ L + 10**
- **C类（偏弱）**：**Q = L + 0 ~ L + 5**

### 3. 到校优先策略（跳过到区搏到校）
触发条件（同时满足）：
- 校内位次**前10%**
- 到校分配校**D ≥ L + 10**
- 到区可报学校**均 < D**

操作：
- 到区填**Q ≥ D + 10**（冲高放弃）
- 到校全力填**D**

---

## 五、三批次联动·量化优先级
### 1. 学校高度强制顺序
**名额到区Q ≥ 名额到校D ≥ 定位校L ≥ 平行志愿保底**

### 2. 全流程量化步骤
1. 用模考位次算出**L分**与定位校
2. 按7+1+7填**平行志愿15个**
3. 用D与L阈值判定**名额到校**
4. 按考生类型确定**名额到区**
5. 校验高度与梯度，锁定志愿

---

## 六、系统量化校验规则
1. **资格拦截**
   - 无到校资格→屏蔽到校志愿
2. **分数校验**
   - Q ≥ D ≥ L，不满足则拦截
3. **梯度校验**
   - 平行志愿必须单调递减
4. **保底校验**
   - 最后1所必须≤L–20（C类≤L–30）
5. **数量校验**
   - 平行15个、到校2个、到区1个

---

## 七、量化规则总表（一页可用）
| 批次 | 数量 | 核心量化公式 | 底线约束 |
|---|---:|---|---|
| 平行志愿 | 15 | 冲刺:L+5~+20；定位:L；保底:L-5~-35 | 填满、递减、保底安全 |
| 名额到校 | 2 | D≥L+5填；D≤L-5不填 | 校内竞争、先录 |
| 名额到区 | 1 | A:L+10~+20；B:L±0~+10；C:L±0~+5 | ≥L、不浪费 |
